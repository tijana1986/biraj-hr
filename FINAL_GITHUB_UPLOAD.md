## 🎯 UPLOAD NA GITHUB - 3 KORAKA:

### KORAK 1: IDES NA GITHUB REPO

```
https://github.com/tijana1986/biraj-hr
```

**Trebam da budeš logovan sa account-om!**

---

### KORAK 2: KLIKNES "Add file"

**U repo stranici, vidis zeleni gumb "Code"**

```
Klikni na njega
Odaberi "Upload files"
```

---

### KORAK 3: DRAG SVE DATOTEKE

**OPCIJA A - Najjednostavnije:**

```
1. Otvoriš Explorer/Finder
2. Vidiš folder: biraj-hr/ sa svim datotekama
3. Select ALL datoteke (Ctrl+A ili Cmd+A)
4. Drag na GitHub upload area
5. GitHub će automatski kreirati folder strukturu!
6. Kliknes "Commit changes"
```

**OPCIJA B - Ako drag-drop ne radi:**

```
1. Klikni "Upload files" gumb
2. Klikni na area gdje piše "Drag files here"
3. Izaberi datoteke sa lista
4. Ili copy-paste jednu po jednu
```
